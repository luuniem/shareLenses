# Main Angular Template

#### 08/20/2016

#### By Niem Nguyen

## Description
This is the template you can use to create a webapp in Angular. It also includes the basic Sass files.

## Installation

- clone repo
- npm install
- bower install
- install globals if needed (gulp, bower, sass, typescript, typescript packages.)
  - npm install gulp -g
  - npm install bower -g
  - gem install sass
  - npm install typescript -g
  - apm install atom-typescript
- gulp build
- gulp serve
